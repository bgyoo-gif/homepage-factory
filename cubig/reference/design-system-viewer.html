Responsive System

4단계 반응형. mobile / tablet / sm-desktop / desktop. wide(1920px) 단계 없음.

Breakpoints
단계 min-width container padding 변수

tablet 768px 32px --ds-container-padding-tablet
sm-desktop 1024px 32px --ds-container-padding-sm-desktop
desktop 1440px 120px --ds-container-padding-desktop
ds-container 규칙
/* 기본 */
.ds-container { width: 100%; margin: 0 auto; padding: 0 16px; }
@media (min-width: 768px) { padding: 0 32px; }
@media (min-width: 1024px) { padding: 0 32px; }
@media (min-width: 1440px) { padding: 0 120px; max-width: 1440px; }
