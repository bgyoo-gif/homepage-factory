## Section 02: Definition

DEFINITION
정의

A restorable workflow is an AI enablement pattern in which sensitive enterprise data is replaced with reversible, structure-preserving representations before AI processing. After the AI model generates outputs, local restoration automatically restores original enterprise data into the results, producing enterprise-ready outputs without manual post-processing. This is the core mechanism by which
복원 가능한 워크플로우는 AI 처리 전에 민감한 기업 데이터를 가역적이고 구조를 보존하는 표현으로 대체하는 AI 활성화 패턴입니다. AI 모델이 결과물을 생성한 후, 로컬 복원이 원본 기업 데이터를 결과물에 자동으로 복원하여 수동 후처리 없이 기업용 결과물을 생성합니다. 이것이 다음의 핵심 메커니즘입니다 —

enables AI adoption on sensitive data.
LLM Capsule이 민감 데이터에서 AI 도입을 가능하게 합니다.

## Section 03: Explanation

Explanation
설명

Traditional data protection workflows are one-directional: protect data, process it, and accept that outputs lack original context. A restorable workflow closes the loop — outputs are automatically restored with real enterprise data. This is what makes
기존 데이터 보호 워크플로우는 단방향입니다 — 데이터를 보호하고, 처리하고, 결과물에 원본 맥락이 없다는 것을 받아들입니다. 복원 가능한 워크플로우는 이 루프를 닫습니다 — 결과물이 실제 기업 데이터로 자동 복원됩니다. 이것이 바로

an AI enabler rather than just a data protector.
LLM Capsule을 단순한 데이터 보호 도구가 아닌 AI 활성화 도구로 만드는 요소입니다.

The key properties of a restorable workflow are: reversible encapsulation (not permanent masking), locally stored mappings (never transmitted), structure-preserving protection (AI accuracy maintained), and deterministic restoration (outputs exactly match original data).
복원 가능한 워크플로우의 핵심 속성은 가역적 캡슐화(영구 마스킹이 아님), 로컬 저장 매핑(전송되지 않음), 구조 보존 보호(AI 정확도 유지), 결정론적 복원(결과물이 원본 데이터와 정확히 일치)입니다.

## Section 04: Example

Example
예시

An HR department uses AI to generate performance review summaries. Employee names, performance ratings, and salary information are encapsulated. AI produces structured summaries from protected data.
인사 부서가 성과 평가 요약을 생성하기 위해 AI를 사용합니다. 직원명, 성과 등급, 급여 정보가 캡슐화됩니다. AI가 보호된 데이터에서 구조화된 요약을 생성합니다.

Restoration restores all employee details, producing manager-ready review documents without any manual editing.
복원이 모든 직원 세부 정보를 복원하여 수동 편집 없이 관리자용 평가 문서를 생성합니다.
