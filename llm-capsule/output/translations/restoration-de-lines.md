# Restoration — Deutsche Übersetzung


## Restoration


Restoration
Wiederherstellung

Restoring AI outputs to original enterprise context using locally stored mappings after AI processing. A core capability of LLM Capsule.
Anreicherung von KI-Ergebnissen mit Originaldaten aus lokal gespeicherten Zuordnungstabellen nach der KI-Verarbeitung. Eine Kernfunktion von LLM Capsule.

Restoration (also known as local restoration) is the process of restoring AI-generated outputs to their original enterprise context. After AI processes encapsulated data, the locally stored mapping between original and protected values is applied to the AI output, replacing encapsulated representations with real enterprise data. AI results are restored locally through this automated process.
Die Wiederherstellung (auch lokale Wiederherstellung) bezeichnet die automatische Anreicherung von KI-Ergebnissen mit den urspruenglichen Unternehmensdaten. Nachdem ein KI-Modell die gekapselten Daten verarbeitet hat, wird die lokal gespeicherte Zuordnung zwischen Original- und Schutzwerten auf die KI-Ausgabe angewendet. Dabei werden geschuetzte Repraesentationen durch echte Unternehmensdaten ersetzt. Dieser Prozess laeuft vollstaendig automatisiert ab.

When LLM Capsule encapsulates a document, it creates a local mapping table. The table records the relationship between each original sensitive value and its replacement. This mapping never leaves the enterprise environment.
Bei der Kapsulierung eines Dokuments erstellt LLM Capsule eine lokale Zuordnungstabelle. Diese Tabelle dokumentiert die Beziehung zwischen jedem vertraulichen Originalwert und seinem Ersatzwert. Die Zuordnung verlässt zu keinem Zeitpunkt die Unternehmensumgebung.

After AI processing completes, the restoration engine applies this mapping in reverse. Encapsulated tokens are replaced with original values. The result is an AI output that contains real names, real figures, and real references, ready for enterprise use.
Nach Abschluss der KI-Verarbeitung wendet die Wiederherstellungskomponente diese Zuordnung in umgekehrter Richtung an. Gekapselte Token werden durch Originalwerte ersetzt. Das Ergebnis ist eine KI-Ausgabe mit echten Namen, Zahlen und Referenzen, die unmittelbar im Unternehmen einsetzbar ist.

Restoration is deterministic. Every encapsulated value maps to exactly one original value. There is no ambiguity, no approximation, and no manual intervention required.
Die Wiederherstellung ist deterministisch. Jeder gekapselte Wert ist genau einem Originalwert zugeordnet. Es gibt keine Mehrdeutigkeit, keine Näherung und kein manuelles Eingreifen.
